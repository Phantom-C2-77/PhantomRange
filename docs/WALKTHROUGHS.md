# Walkthroughs

Walkthroughs are not included in the public repo to prevent cheating.

If you're stuck, here are some general tips:
- Use browser DevTools to inspect requests and responses
- Try common attack payloads from OWASP cheat sheets
- Check for hidden endpoints, parameters, and cookies
- Use tools like Burp Suite, curl, sqlmap, and ffuf
- Read the vulnerability descriptions on the /vulns page for clues
- Submit flags via: `curl -X POST http://localhost:9000/flag -H 'Content-Type: application/json' -d '{"flag":"FLAG{...}"}'`

Happy hacking! 🚩
